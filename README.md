## 陈默的博客

> 3分钟快速搭建一个轻量级博客.

## 特性

- 无需构建，写完文档直接发布
- 容易使用并且轻量 (压缩后 ~21kB)
- 智能的全文搜索
- 提供多套主题
- 丰富的 API
- 支持 Emoji
- 兼容 IE11

## 技术

- 本站使用docsify快速构建，方便快捷高效

## 快速开始

- 安装 `docsify-cli` 工具：

```javascript
npm i docsify-cli -g
```

- 初始化项目：

```javascript
docsify init ./docs
```

- 预览网站：

```javascript
docsify serve docs
```

## 文件说明

> 本站发布目录在docs下

```bash
docs			#网站发布目录
copy2docs.bat 	#快速拷贝网站文档进入docs目录
EXCLUDE.txt 	#需要排除拷贝的文件和目录
localRun.bat 	#本地快速启动预览网站命令
```
